Lessons in
4th Quarter

  1. Javascript Events
    • What’re Events? - These are actions that are triggered either by mouse click, function calls, keyboard strokes, etc.
    • Describes how HTML elements are to be displayed on media
    • Functions: Remains a block of text unless it’s called by an event handler.
    • Event Handlers: You can respond to an event on your page using these.
    • Popup Boxes: Windows that pop out of the screen, useful in warning the user, gathering input, or verification.
    • browser sample

  2. Conditions, Controls, and Loops
    • Conditional Statements: perform different actions for different decisions. If Else If Statement: refers to an if statement depending on another if statement.
    • If Statement: used to check or verify a condition & is executed ONLY if the condition is true
      (Should be referred as a statement, not a function.) This construct evaluates a condition to True or False, then runs a specific code depending on the result.
    • Nested If Statement: If statement w/ another if statement (parent and child);
      used when “child” condition is only checked when the parent condition is true.
    • If-Else Statement: similar to if statement, but we are given an alternative instruction when the argument is NOT true.
    • If Else If Statement: refers to an if statement depending on another if statement.
    • browser sample

  3. Conditions, Controls, and Loops
    • Switch Statement: used when a condition may have multiple results &
      a different set of operation is done based on each result or input.
    • Functions: a self-contained piece of code that performs a particular “function” when called.
    • Argument: When functions are defined using “function,” followed by its name and the open & close
      parentheses; provides additional information needed by the function to process.
    • Return Statement: For when a function requires specifying a resulting output (ex: computation); functions that give out a value must use this
    • browser sample

  4. Loops
    • The for loop is used when you know how many times the block of code must be iterated
    • The while loop is used if you know when the condition will be met.
      It will loop through the set of codes until the condition is met.
    • Do-while loops work like the while loop except that the code will be executed once and is repeated as
      long as the set condition is true, but will stop once the condition is false.
    • For-in loop through the elements of an array or an object.
    • The break statement will stop or break the loop once the specified condition becomes true and
      then execute the nextcode if any after the loop.
    • The continue statement will stop or break the loop for a setvalue then continue to
      loop with the nest value.
    • browser sample

Reflection

This quarter's JavaScript lessons were a big step up for me. I learned about events, which make websites interactive by responding to clicks and other actions. I also got the hang of conditional statements and functions, which help control how code runs and make it easier to reuse.

At first, it was tough, but with practice, I got better at using these tools in my projects. I can now make web pages that react to what users do and write code that's more organized and efficient.

Overall, it was a great learning experience, and I'm excited to keep improving my skills. These lessons have given me a solid foundation for building more complex and engaging web applications in the future.
